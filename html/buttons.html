A Button Library

1. Square, Depressible

I first saw this minimalist button on gov.uk, finding its skeuomorphic downward 'movement' quite satisfying.

This effect is achieved as follows: 


#button-one {
    border: solid black 1px;
    border-bottom: 6px black solid;
}

#button-one:active {
    border-bottom: 2px black solid;
    transform: translateY(2px);
}

Take care when changing live border values such as with :hover or :active, as this can bump neighbouring block elements around.

2. Square, Pop-up on Hover, Depressible

This adaptation of button distinguishes a hovered button from its neighbours by breaking their shared boundary (it sticks out).

While the pop-up behaviour is satisfying for desktop mouse-users, mobile and keyboard-only users inability to hover means they wont encounter value here.

Combining the pop-up and disabled pop-up behaviour with colour coding leads to a less jarring result:

This is achieved as follows: 


#button-two {
    border: solid black 1px;
    border-bottom: 2px black solid;
}

#button-two:hover {
    border-bottom: 6px black solid;
    transform: translateY(-2px);
}

#button-two:active {
    border-bottom: 2px black solid;
    transform: translateY(0px);
}

#button-two:disabled {
    border-bottom: 2px black solid;
    background-color: lightgrey;
    color: #666;
    transform: translateY(0px);
}
