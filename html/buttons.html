A Button Library

1. Square, Depressible

I first saw this minimalist button on gov.uk, finding its skeuomorphic downward 'movement' quite satisfying.

This effect is achieved as follows: 


#button-one {
    border: solid black 1px;
    border-bottom: 6px black solid;
}

#button-one:active {
    border-bottom: 2px black solid;
    transform: translateY(2px);
}

Take care when changing live border values such as with :hover or :active, as this can bump neighbouring block elements around.

2. Square, Presents on Hover, Depressible

An adaptation of gov.uk's design which distinguishes a hovered button from a group by breaking the group's shared boundary (or 'sticking out').
The expectation of a 'pop-up' effect for active buttons may also be used to distinguish disabled buttons.

While this is satisfying for desktop mouse users, it goes without saying that these button effects have no value to mobile and keyboard-only users.

Nonetheless, lets combine with some colour coding...

This is achieved as follows: 


#button-two {
    border: solid black 1px;
}

#button-two:hover {
    border-bottom: 6px black solid;
    transform: translateY(-2.5px);
}

#button-two:active {
    border-bottom: 2px black solid;
    transform: translateY(-0.5px);
}

#button-two:disabled {
    border: solid black 1px;
    background-color: lightgrey;
    color: #666;
    transform: translateY(0px);
}
